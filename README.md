# GERMAN_Driver

## Проект Олега Герман

Этот каталог содержит драйверы для различного измерительного оборудования, предназначенные для использования на платформе MOKO SE по автоматизации измерений.

### Содержимое

*   **DriverCompiled_DLL**: Скомпилированные версии драйверов (DLL) для различных приборов.
*   **Drivers**: Исходные файлы драйверов в формате LabVIEW (.vi), сгруппированные по производителям оборудования.
*   **Icon**: Иконки, используемые в проекте.
*   **PythonDocs**: Python-обертки для взаимодействия с драйверами.
*   **installer**: Установочные файлы для развертывания драйверов.
*   **iss**: Скрипты Inno Setup для создания инсталляторов.
*   **Проектные файлы LabVIEW**: Файлы проекта LabVIEW (`.lvproj`, `.lvlps`).

### Назначение

Основная цель этого проекта — предоставить стандартизированный набор драйверов для автоматизации работы с измерительными приборами в среде MOKO SE.
